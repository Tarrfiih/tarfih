# Tarfih Application

Tarfih is a client-side web application designed to help users explore tourism, entertainment, and leisure activities across top Algerian coastal and desert cities like Annaba, Algiers, Oran, and more.

## Features
- **AI Smart Search**: Utilize an NLP-based search to filter activities (e.g. price limits, group types like families, or rating priorities).
- **Favorites System**: Clients can "heart" their favorite locations, retaining them via local storage. 
- **Booking Flow**: Smoothly animated modals to simulate premium appointment booking success states and form validations.
- **Categorization**: Filters for Sea, Sahara, Sports, Hiking, and Historical sites.

## Usage
Simply load `index.html` in your browser. All interactions and mock data are provided inside `script.js`, built with modularity and design aesthetics in mind using modern CSS and native JS.
